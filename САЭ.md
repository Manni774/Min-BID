Описание проекта информационной системы
Данная ИС разработана для учёта земельных участков и арендных соглашений в контексте работы Управления информационно-аналитического сопровождения соответствующего Министерства.
Цель работы:
Разработка и сопровождение информационной системы для учёта земельных участков и договоров аренды.
Технологический стек:
Платформа: .NET
СУБД: Microsoft SQL Server
